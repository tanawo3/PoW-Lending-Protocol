# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-17

### Added
- Initial release of PoW Lending Protocol on GenLayer.
- Proof of Work GitHub payload evaluation via GenVM.
- UI Dashboards for Borrowers, Lenders, and Predictors.
- ABI Encoding standard implemented via `useGenLayer`.
- Trace Fallback mechanism with recursive JSON extraction.
- Deterministic random number generation.
